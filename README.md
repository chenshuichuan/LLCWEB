# LLCWEB
## service 
1. 接口名称：DocumentService.findAll() <br>
   接收参数：文档类、页码与每页访问量 <br>
   返回参数：文档集合 <br>
   功能：将传入的文档对象拥有的属性拼接到搜索条件中，查询数据库匹配记录；<br>
         将查询结果按日期排序，返回符合条件的文档集合 <br>
